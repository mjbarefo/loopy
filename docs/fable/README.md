# Fable Prompts

This directory contains the evolution of **loopy** through a series of Fable prompts.

- [FABLE_PROMPT.md](FABLE_PROMPT.md) — Initial mission to build loopy's core skeleton and loop engine.
- [FABLE_PROMPT_2.md](FABLE_PROMPT_2.md) — Second session focused on the monitor, judgment system, and loop lifecycle.
- [FABLE_PROMPT_3.md](FABLE_PROMPT_3.md) — Production pass to take loopy from a project to a production-quality v0.1.
- [FABLE_PROMPT_4.md](FABLE_PROMPT_4.md) — Identity pass to align the working monitor with the visual design of the launch splash.
- [FABLE_PROMPT_5.md](FABLE_PROMPT_5.md) — The "calm pass" to restore minimalist taste and intuitive design inspired by herdr.
