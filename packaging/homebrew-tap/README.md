# mjbarefo/homebrew-tap

Homebrew formulae for [loopy](https://github.com/mjbarefo/loopy).

```sh
brew install mjbarefo/tap/loopy
```

Only stable releases land here — release candidates are GitHub prereleases
on the loopy repository and are never published to this tap.

`Formula/loopy.rb` is generated by loopy's release workflow
(`scripts/homebrew-formula.sh` there) and updated automatically when a
stable release publishes: the release fires a `repository_dispatch` event
(`loopy-release`) and the update workflow here downloads the formula asset
from that release and commits it. Don't edit the formula by hand; fix the
generator upstream.
